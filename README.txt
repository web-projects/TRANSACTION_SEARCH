USAGE INSTRUCTIONS

1. Place a log file (i.e. log230607.txt) in the Executable/in directory.
2. Update Executable/appsettings.json FileGroup->Input with the log filename from #1.
3. If needed, update the "Request" type to narrow down the filter:
   options:
    "GetPayment"
    "GetPayment"
    "GetCreditOrDebit"
    "GetPIN"
    "GetZIP"
    "GetVerifyAmount"
    "RemoveCard"
    "DeviceUI"
3. Run the program Executable/TRANSACTION_SEARCH.exe
4. Output segments in Executable/in
